COMMON QUESTIONS

A few things to know

Is Bead Grid Studio free?

Yes. The editor runs entirely in your browser with no account or subscription.

Where is my pattern saved?

Your current project autosaves to this browser’s local storage. Export JSON for a portable backup.

Can I use the PNG as a crafting guide?

Yes. Export with a white background for printing or transparency for use in other designs.

Which grid size should I choose?

16×16 is great for small motifs, 24×24 is a flexible default, and 32×32 allows more detail.
