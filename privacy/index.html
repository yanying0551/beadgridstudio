Bead Grid Studio

PRIVACY

Your pattern stays in your browser.

Effective July 16, 2026. Bead Grid Studio is currently a browser-only tool with no accounts or application backend. It does not send your pattern to a Bead Grid Studio server.

Project storage

The editor uses your browser’s localStorage to autosave the current project. This data stays in that browser on that device until it is overwritten or you clear the board or site data.

Files you choose

JSON imports are read locally in your browser. JSON and PNG exports are generated locally and downloaded to your device. We do not receive those files.

Network requests

Loading the Studio requests the Inter font from Google Fonts, so your browser connects to Google to retrieve it. That request is governed by Google’s privacy practices. The site currently has no advertising or analytics integration.

Your choices

  • Clear the board or your browser’s site data to remove the local autosave.
  • Export JSON before clearing data if you want a portable backup.
  • Block third-party font requests using your browser or content blocker.

Changes

If cloud storage, accounts, analytics, or other network features are added, this notice will be updated before those features are represented here.
