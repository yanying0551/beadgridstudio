## Positioning

### Title

Bead Grid Studio — Free Fuse Bead Pattern Maker

### One-line description

Design fuse bead patterns and pixel art in your browser, then save your project or export a crisp PNG guide for free.

### Short description

A fast, free, private-in-your-browser grid editor for drawing fuse bead patterns, testing symmetry, counting colors, and exporting craft-ready guides.

### Long description

Bead Grid Studio is a free browser-based editor for fuse bead patterns and pixel art. Paint on a 16×16, 24×24, or 32×32 grid; choose from a practical color palette; mirror designs horizontally or vertically; undo and redo edits; and export a high-resolution PNG. Projects autosave in the current browser, and JSON import/export provides a portable backup. No account or subscription is required.

## Feature facts approved by the current implementation

- 16×16, 24×24, and 32×32 grids
- Click/drag and keyboard-operable painting
- Erase, clear, undo, and redo
- Horizontal and vertical mirror modes
- Color counts
- Browser-local autosave
- JSON import/export
- High-resolution PNG export
- No account or subscription
